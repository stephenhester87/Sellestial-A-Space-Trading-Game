# Sellestial
A dystopian text-based space trading game focused on strategy, economy, and player choice.

Overview
--------

In Sellestial, the player starts with limited money and fuel, and a basic ship that can carry only one item at a time. Each region in the game offers the same items, but at different prices, creating opportunities for strategic trading.

The goal is to make smart buying and selling decisions while accounting for fuel costs and cargo limitations in order to grow your wealth.

Features
--------

- Wallet system to track player money
- Inventory system with cargo capacity limits
- Multiple regions with different item prices
- Buy and sell trading mechanics
- Fuel system for travel between regions
- Ship upgrade system to increase cargo capacity
- Profit-based gameplay strategy
  

Gameplay
--------

- Start with a set amount of money and fuel
- View market prices in each region
- Buy items where they are cheap
- Travel to another region (costs fuel)
- Sell items for profit
- Upgrade your ship to carry more items
- Repeat and grow your wallet and buy better ships.


How to play
--------
- Simply run the main.py in your dedicated terminal
